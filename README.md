# BARO Admin

Panel web para que las barberias registren y gestionen sus datos, servicios, barberos y citas.

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue en Render

- Tipo: Static Site
- Build Command: `npm install && npm run build`
- Publish Directory: `dist`
- Environment Variable: `VITE_API_BASE_URL=https://baro-api.onrender.com`
